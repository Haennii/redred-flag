# RedFlag — 은행 재무제표 이상징후 분석 대시보드

4대 시중은행(KB국민·신한·하나·우리)의 재무제표를 분석하여 감사위험을 조기에 탐지하는 대시보드입니다.

## 개요

회계법인 감사부문에서 실제로 사용하는 관점으로 은행 재무지표의 이상징후를 자동 탐지하고 시각화합니다.  
공식 규제 기준이 존재하는 지표는 해당 기준을 적용하고, 나머지는 **ISA 520 분석적 절차**에 따라 동종 4행 평균 대비 편차로 이상징후를 판단합니다.

## 탐지 지표 및 적용 기준

|------|-----------|-----------|-----------|

| **자기자본이익률 (ROE)** | 동종 4행 평균 대비 편차 (%, 주의 −10%p) | ISA 520 분석적 절차 · 금감원 국내은행 경영현황 공시 | ROE 하락 시 자본 효율성 저하 또는 이익 과대계상 수정 가능성. 지속 하락은 이익조정 유인 증가 신호 |

| **비이자이익** | 전년 대비 증감률 — 동종 4행 평균과의 편차 (pp, 주의 −10pp) | ISA 520 분석적 절차 · 금감원 국내은행 경영현황 공시 | 수익 다변화 약화, 금리 변동 취약성 증가. 유가증권 평가손 집중 여부 점검 |
| **대손비용** | 전년 대비 증감률 — 동종 4행 평균과의 편차 (pp, 주의 +15pp) | ISA 520 분석적 절차 · 금감원 국내은행 경영현황 공시 | 동종 대비 급증은 전년도 충당금 과소 적립 수정 신호. IFRS 9 ECL 모델의 PD·LGD 가정 및 시나리오 가중치 적정성 점검 |
| **대손비용률** | 동종 4행 평균 대비 편차 (%, 주의 +20%p) | ISA 520 분석적 절차 · 금감원 국내은행 경영현황 공시 | 대출 포트폴리오 자산건전성 선행지표. 동종 상회 시 부문별(가계·기업) 대출 분포 및 담보 적정성 감사 표본 확대 |

### 편차 판정 기준 (ISA 520 동종 4행 비교)

| 구분 | 주의 (Watch) | 경고 (Warning) | 위험 (Danger) |
|------|-------------|----------------|--------------|
| 수준 지표 (낮을수록 나쁨: ROE·NIM·대손비용률 제외) | −10% | −20% | −35% |
| 수준 지표 (높을수록 나쁨: 대손비용률) | +20% | +40% | +80% |
| 증감률 지표 (낮을수록 나쁨: 순영업수익·비이자이익) | −10pp | −20pp | −35pp |
| 증감률 지표 (높을수록 나쁨: 대손비용) | +15pp | +30pp | +60pp |

> 편차 = (개별 은행 값 − 4행 평균) / |4행 평균| × 100 (%)

## 기술 스택

- **Frontend**: React + TypeScript + Vite
- **차트**: Recharts
- **스타일**: Tailwind CSS
- **데이터**: DART 전자공시 OpenAPI (2023-2024 실데이터) + 사업보고서 기반 추정치 (2020-2022)

## 실행 방법

```bash
# 의존성 설치
npm install

# DART API 키 설정 (opendart.fss.or.kr 무료 발급)
cp .env.example .env
# .env 파일에 VITE_DART_API_KEY 입력

# 개발 서버 실행
npm run dev
```

> API 키가 없으면 사업보고서 공시 기반 데이터로 자동 동작합니다.

## 데이터 출처

- DART 전자공시시스템 (opendart.fss.or.kr) — 별도재무제표 (fnlttSinglAcntAll)
- 각 은행 사업보고서 — 보통주자본비율(CET1) 등 감독당국 보고 지표
- 금융감독원 금융통계정보시스템 — 업계 평균 벤치마크

## 개발 과정에서 발견한 것들

### 배경

회계법인 감사부문 지원을 준비하면서, 감사인이 실제로 재무제표를 볼 때 어떤 관점으로 이상징후를 포착하는지를 직접 구현해보고 싶었습니다. 단순히 지표를 보여주는 것에 그치지 않고, **어떤 기준으로, 왜 이것이 이상한가**를 설명할 수 있는 시스템을 목표로 했습니다.

### 문제 1 — 은행마다 같은 항목을 다른 이름으로 공시한다

금융감독원 DART에서 4개 은행의 재무제표를 불러오다 보니, 같은 개념의 항목이 은행마다 다른 이름으로 기재되어 있었습니다.

| 항목 | KB국민 | 신한 | 하나 | 우리 |
|------|--------|------|------|------|
| 대손비용 | `신용손실충당금 전입액` | `신용손실충당금전입` | `신용손실충당금전입액` | `신용손실충당금전입액` |
| 수수료이익 | `순수수료이익` | `순수수료손익` | `순수수료이익` | `순수수료이익` |
| 트레이딩손익 | `당기손익-공정가치측정 금융상품 순손익` | `당기손익-공정가치측정금융상품관련손익` | `순당기손익-공정가치측정금융상품이익` | `당기손익-공정가치측정금융상품관련손익` |
| 기타영업손익 | 단일 계정 | 단일 계정 | **수익·비용 분리 공시** | 단일 계정 |

신한은행의 경우, 대손비용 계정명 끝에 '액'이 붙지 않아 시스템이 해당 항목을 아예 인식하지 못했습니다. 그 결과 신한의 대손비용이 0원으로 잡혔고, 이를 기준으로 계산한 감사위험 점수가 실제와 전혀 다른 값으로 출력되었습니다.

하나은행은 기타영업손익을 하나의 숫자로 공시하지 않고 '기타영업수익'과 '기타영업비용'으로 나누어 공시하고 있어, 이 둘을 차감하는 별도 처리가 필요했습니다. 우리은행의 트레이딩손익(당기손익-공정가치측정금융상품관련손익)은 15,091억원 규모였는데, 계정명이 달라 인식되지 않으면서 비이자이익이 실제 +2,628억원이 아닌 −12,465억원으로 계산되는 오류가 발생했습니다.

**왜 이것이 중요한가**

실제 감사에서 분석적 절차(ISA 520)를 수행할 때도 이와 동일한 문제가 발생합니다. 감사인이 전년도 또는 동종 업계 수치와 비교하려면, 먼저 그 수치가 같은 기준으로 집계된 것인지를 확인해야 합니다. 계정 분류 방식이나 공시 형식의 차이를 파악하지 않은 채 숫자만 비교하면, 없는 이상징후를 발견했다고 착각하거나 실제 이상을 놓칩니다. 이 과정을 직접 경험하면서, 데이터의 정합성을 확보하는 작업이 분석보다 선행되어야 함을 체감했습니다.

### 문제 2 — 이상징후의 기준을 어디서 가져오는가

처음에는 "ROE가 8% 미만이면 위험"처럼 고정된 기준치를 사용했습니다. 그런데 이 기준이 어디서 나온 것인지, 감사 실무에서 실제로 사용하는 방식인지 확인하기 어려웠습니다.

조사를 통해 두 가지 접근법으로 정리했습니다.

- **공식 규제 기준이 있는 지표**: 보통주자본비율(CET1)은 Basel III 및 금융위원회 고시에 의해 4대 은행 기준 최저 8.0%가 명확히 규정되어 있습니다. 이 기준은 그대로 적용했습니다.
- **공식 기준이 없는 지표**: ROE, NIM, 대손비용 등은 법령으로 정해진 임계값이 없습니다. 이 경우 ISA 520(분석적 절차)에 따라 동종 업계 평균과의 편차를 기준으로 삼는 것이 감사 실무에서 일반적인 접근 방식입니다. 4개 은행 평균을 기준선으로 설정하고, 개별 은행이 평균 대비 얼마나 벗어나는지를 백분율 편차로 계산했습니다.

### 최종 분석 결과 — 신한은행 위험 판정 (80/100)

계정명 오류를 모두 수정한 뒤 분석한 결과, **신한은행이 4개 은행 중 유일하게 위험(80점) 판정**을 받았습니다.

주요 원인은 비이자이익의 급격한 하락입니다. 2024년 신한은행의 비이자이익은 −909억원으로, 4행 중 유일하게 마이너스로 전환되었습니다. 전년 대비 −224% 감소로, 4행 평균 감소율(−89.2%)과의 편차가 −134.9%p에 달했습니다. 세부 원인을 분해하면, 수수료수익(+10,002억원)과 트레이딩손익(+2,781억원)이 있었음에도 기타영업손익이 −13,693억원으로 이를 모두 상쇄했습니다.

실제 감사라면 이 시점에서 다음 절차로 이어집니다. 기타영업손익의 구성 항목(파생상품 평가손실, 유가증권 관련 손익, 기타 충당금 등)을 세부적으로 분석하고, 전년과의 변동이 경제적 실질을 반영한 것인지, 아니면 회계 처리 방식의 변경이나 일회성 항목이 반영된 것인지를 확인하는 절차가 필요합니다.

반면 **우리은행은 0점(양호)**으로 이상징후가 없었습니다. 2024년 약 6,983억원의 대손충당금 환입이 발생하면서 대손비용이 마이너스로 전환되었고, 이것이 4행 평균을 끌어내려 우리은행의 대손 관련 지표가 모두 양호하게 나타난 것입니다. 이처럼 동종 평균 자체가 특정 은행의 이례적인 수치에 영향을 받을 수 있다는 점도 분석적 절차의 한계로 인식해야 합니다.
